# MCP Starter Kit

Learning resources for building Model Context Protocol (MCP) servers. Three guides covering MCP fundamentals, a complete implementation walkthrough, and a step-by-step tutorial project.

## Contents

| File | Description |
|------|-------------|
| `mcps-101.md` | MCP fundamentals — what it is, how it works, protocol overview |
| `mcp-complete-implementation-guide.md` | Full implementation guide — Python & TypeScript templates, transport protocols, security, deployment, common pitfalls |
| `lazycalculator.md` | Step-by-step tutorial — build a joke calculator MCP server from scratch (see [lazy-calculator-mcp](https://github.com/Tommy2Face/lazy-calculator-mcp) for the finished project) |

## Quick Start

These are reference docs — no installation needed. Start with `mcps-101.md` for concepts, then follow `lazycalculator.md` to build your first server.
